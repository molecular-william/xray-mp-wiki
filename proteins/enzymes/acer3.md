# Alkaline Ceramidase 3 (ACER3)

## Overview

Alkaline ceramidase 3 (ACER3) is a human integral membrane protein with a seven-transmembrane (7TM) domain architecture that belongs to the progesterone and adipoQ receptor (PAQR) family. [ACER3](/xray-mp-wiki/proteins/acer3/) functions as an intramembrane ceramidase that hydrolyzes [Ceramide](/xray-mp-wiki/reagents/lipids/ceramide/) into [Sphingosine](/xray-mp-wiki/reagents/lipids/sphingosine/) and free fatty acids. The structure was determined by serial synchrotron crystallography (SSX) using the CrystalDirect automated platform for in meso crystallization.

## Biological / Functional Insights

### Intramembrane ceramidase activity

[ACER3](/xray-mp-wiki/proteins/acer3/) is an intramembrane ceramidase that hydrolyzes [Ceramide](/xray-mp-wiki/reagents/lipids/ceramide/) into [Sphingosine](/xray-mp-wiki/reagents/lipids/sphingosine/) and a free fatty acid. Like other PAQR family members, it contains a conserved active site with three histidines coordinating a zinc ion within the transmembrane domain.
