# n-Dodecyl-beta-D-Maltoside (DDM)

## Overview

n-Dodecyl-beta-D-maltoside (DDM) is a mild nonionic detergent widely used in membrane protein purification and crystallization. It was used to solubilize and crystallize TmYidC, enabling the first structure with the N-terminal amphipathic helix resolved. The shorter-chain decyl maltoside (DM) was noted as harsher and possibly disrupting these flexible regions.

## Use in Membrane Protein Work

### TmYidC crystallization

DDM was used as the detergent for TmYidC purification and crystallization. The structure revealed that DDM preserved the N-AH and C1 regions, unlike DM which had been used in previous TmYidC structures where these regions were not visible.
