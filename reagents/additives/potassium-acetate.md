# Potassium acetate

## Overview

Potassium acetate (CH₃COOK) is a potassium salt used as a crystallization additive and buffer component. In the context of P-type ATPase crystallization, it provides the K⁺ ions required for the E2·MgF₄²⁻·2K⁺ state.

## Properties


## Use in Membrane Protein Work

### Crystallization buffer for E2·MgF₄²⁻·2K⁺ state

100 mM potassium acetate used in crystallization buffer for shark Na+,K+-ATPase
in the E2·MgF₄²⁻·2K⁺ state. Provides K⁺ ions required to occupy the two
transmembrane cation-binding sites.
