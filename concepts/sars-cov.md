# SARS-CoV

## Overview

SARS-CoV (Severe Acute Respiratory Syndrome Coronavirus) is the original SARS coronavirus (2002–2003) whose spike protein receptor-binding domain (RBD) was the first coronavirus RBD structure solved. It binds to the same host receptor as SARS-CoV-2: ACE2. The SARS-CoV RBD structure (PDB: 2AJF) served as the search model for molecular replacement in solving the SARS-CoV-2-CTD/hACE2 complex structure.
