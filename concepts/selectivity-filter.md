# Ion Channel Selectivity Filter

## Overview
The selectivity filter is a narrow region of the ion channel pore that determines which ions can pass through. It is formed by conserved amino acid sequences that create specific coordination geometries for target ions. The architecture and chemistry of the selectivity filter differ between sodium, potassium, and calcium channels, reflecting their distinct physiological roles.
