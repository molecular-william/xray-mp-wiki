# LeuT Return State Mechanism

## Overview
The [LEUT](/xray-mp-wiki/proteins/enzymes/leut/) return state mechanism describes the inward-to-outward transition of the [Neurotransmitter/Sodium Symporter (NSS) Family](/xray-mp-wiki/concepts/nss-family/) transporter [LEUT](/xray-mp-wiki/proteins/enzymes/leut/) in the absence of both Na+ and substrate. This return transition is the rate-limiting step of the NSS transport cycle. The key structural feature is that the conserved Leu25 residue of the unwound TM1 region rotates into the empty substrate-binding pocket (S1), compensating for the absence of substrate and Na+ ions. This state is proposed to be an H+-occluded state where Glu290 is protonated and neutral.
