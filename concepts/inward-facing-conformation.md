# Inward-Facing Conformation of ABC Transporters

## Overview
The inward-facing conformation is a structural state of ABC transporters in which the transmembrane domain opens toward the cytoplasm while the periplasmic gate is closed. This conformation represents the post-translocation state in the transport cycle, following substrate release into the cytoplasm.

## Mechanism/Details
The inward-facing conformation is a structural state of ABC transporters in which the transmembrane domain opens toward the cytoplasm while the periplasmic (or extracellular) gate is closed. This conformation represents the post-translocation state in the transport cycle, following substrate release into the cytoplasm. The BhuUV-T structure from Burkholderia cenocepacia (3.2 Å) provided the first high-resolution structure of a type II ABC transporter in the inward-facing conformation, in the nucleotide-free and substrate-free form. The cytoplasmic gate is completely open, formed by separation of hydrophobic residues (L177, L178, I181) of TM5 from the other monomer. The periplasmic gate is tightly sealed by R204-D200 salt bridges and hydrophobic interaction (L203-L203) in helix H5a.
