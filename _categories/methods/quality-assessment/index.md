# Quality Assessment Methods

## Overview

Quality assessment methods evaluate protein purity, concentration, and integrity at various stages of the workflow.

## Methods

### Coomassie Staining
- **Description**: Protein visualization using Coomassie Brilliant Blue dye
- **Advantages**: Sensitive (~10-100 ng protein), compatible with SDS-PAGE
- **Disadvantages**: Requires destaining, semi-quantitative
- **Page**: [[coomassie-staining]]
