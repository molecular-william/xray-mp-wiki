# Structure Determination Methods

## Overview

Structure determination methods reveal the three-dimensional arrangement of atoms in macromolecules.

## Methods

### X-ray Crystallography
- **Description**: Uses X-ray diffraction from crystallized samples
- **Advantages**: High resolution (often < 2 Å), well-established
- **Disadvantages**: Requires crystals, radiation damage
- **Page**: [[xray-crystallography]]

### Cryo-EM (Cryogenic Electron Microscopy)
- **Description**: Electron microscopy of frozen-hydrated samples
- **Advantages**: No crystals needed, handles large complexes
- **Disadvantages**: Lower resolution for smaller proteins, expensive equipment
- **Page**: [[cryoem]]

### Molecular Replacement
- **Description**: Phasing method using known homologous structure
- **Advantages**: Fast, doesn't require heavy atoms
- **Disadvantages**: Requires homologous structure, can have model bias
- **Page**: [[molecular-replacement]]

### Serial Femtosecond Crystallography (SFX)
- **Description**: X-ray free-electron laser method using microcrystals
- **Advantages**: Avoids radiation damage, works with small crystals
- **Disadvantages**: Requires XFEL facility, complex data analysis
- **Page**: [[serial-femtosecond-crystallography]]
