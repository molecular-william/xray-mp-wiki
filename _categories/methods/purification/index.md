# Purification Methods

## Overview

Purification separates the target protein from other cellular components. Multiple steps are typically required to achieve high purity.

## Methods

### Affinity Chromatography
- **Description**: Uses specific binding interactions (e.g., His-tag/Ni-NTA)
- **Advantages**: High specificity, one-step purification possible
- **Disadvantages**: Requires tag, may need tag removal
- **Page**: [[affinity-chromatography]]

### Size-Exclusion Chromatography (SEC)
- **Description**: Separates by molecular size
- **Advantages**: Gentle, removes aggregates, buffer exchange
- **Disadvantages**: Low resolution for similar-sized proteins, sample dilution
- **Pages**: [[size-exclusion-chromatography]], [[superdex-columns]]

### Ion-Exchange Chromatography
- **Description**: Separates by charge
- **Advantages**: High resolution, scalable
- **Disadvantages**: Requires optimization, can denature proteins

### Concentration
- **Amicon Ultra Filters** - Concentrate using centrifugal filtration
- **Page**: [[amicon-filters]]

### Centrifugation
- **Description**: Separation by density using centrifugal force
- **Applications**: Membrane pellet recovery, differential centrifugation
- **Page**: [[centrifugation]]

### Dialysis
- **Description**: Buffer exchange using semi-permeable membrane
- **Advantages**: Gentle, removes small molecules
- **Disadvantages**: Slow, requires large buffer volumes
- **Page**: [[dialysis]]

### IMAC (Immobilized Metal Affinity Chromatography)
- **Description**: Uses metal-chelating resins for His-tag purification
- **Resins**: Ni-NTA, TALON Cobalt Resin
- **Pages**: [[imac]], [[talon-resin]]
