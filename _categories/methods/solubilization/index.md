# Solubilization Methods

## Overview

Solubilization extracts membrane proteins from lipid bilayers using detergents or other amphipathic molecules.

## Methods

### Detergent Solubilization
- **Description**: Using detergents to form protein-detergent micelles
- **Key considerations**: Detergent CMC, concentration, protein stability
- **Pages**: [[solubilization]], [[ddm]], [[lmng]], [[mng-detergent]], [[og]]

### Lipidic Cubic Phase (LCP)
- **Description**: Embedding membrane proteins in lipid mesophase
- **Advantages**: Native-like environment, no detergent needed
- **Disadvantages**: More complex, limited to certain proteins
- **Page**: [[lipidic-cubic-phase]]
