# Methods

## Overview

This wiki documents methods and protocols for X-ray crystallography of membrane proteins, covering the entire workflow from expression to structure determination.

## Method Categories

### Cell Lysis
- **Microfluidizer** - High-pressure cell disruption
- **Page**: [[microfluidizer]]

### Crystallization
- **Lipidic Cubic Phase (LCP)** - Lipid-based crystallization for membrane proteins
- **Vapor Diffusion** - Most widely used crystallization method
- **Microbatch** - Oil-overlay crystallization method
- **Pages**: [[lipidic-cubic-phase]], [[vapor-diffusion]], [[microbatch]]

### Expression Systems
- **Baculovirus Expression** - Sf9 insect cell expression system
- **HEK293 Cells** - Mammalian cell expression
- **Pichia pastoris** - Yeast expression system
- **Sf9 Cells** - Insect cell expression
- **Pages**: [[baculovirus-expression]], [[hek293-cells]], [[pichia-pastoris]], [[sf9-cells]]

### Purification
- **Affinity Chromatography** - Tag-based purification
- **Amicon Ultra Filters** - Concentration and buffer exchange
- **Centrifugation** - Separation by density
- **Dialysis** - Buffer exchange and equilibration
- **IMAC** - Immobilized Metal Affinity Chromatography
- **Size-Exclusion Chromatography (SEC)** - Separation by size
- **Superdex Columns** - SEC resin types
- **TALON Cobalt Resin** - Cobalt-based affinity resin
- **Pages**: [[affinity-chromatography]], [[amicon-filters]], [[centrifugation]], [[dialysis]], [[imac]], [[size-exclusion-chromatography]], [[superdex-columns]], [[talon-resin]]

### Quality Assessment
- **Coomassie Staining** - Protein visualization
- **Page**: [[coomassie-staining]]
