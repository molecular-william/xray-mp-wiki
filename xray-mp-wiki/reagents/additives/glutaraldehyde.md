# Glutaraldehyde

## Overview

Glutaraldehyde is a bifunctional crosslinking agent commonly used in protein crystallography for stabilizing protein crystals and for cryo-protection. In the AcrB multidrug efflux transporter study, crystals were harvested and cryo-protected using solutions containing glycerol, with glutaraldehyde potentially used as a fixative in sample preparation steps.


## Properties

- **Chemical name**: glutaraldehyde
- **Chemical formula**: C5H8O2
- **Class**: bifunctional crosslinker
