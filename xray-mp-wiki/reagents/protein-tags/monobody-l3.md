# Monobody L3

## Overview

Monobody L3 is an engineered fibronectin type III domain (FN3) protein selected from diversified phage-display libraries as a crystallization chaperone for the fluoride channel (Fluc) from Bordetella pertussis (Bpe). L3 is a strong channel blocker with sub-micromolar affinity that completely blocks fluoride current upon complexation. Block events by L3 typically last tens of seconds in single-channel recordings. L3 was used as an internal standard (100% block level) in electrophysiology experiments comparing S8 partial block behavior.
